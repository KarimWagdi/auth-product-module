# auth-product-module
```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
# or run
$ docker-compose up

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

$ npm run test

# test coverage
$ npm run test:cov
